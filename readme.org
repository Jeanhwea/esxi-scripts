* 参考命令实例
** 查看有哪些虚机
   #+BEGIN_SRC sh
     vim-cmd vmsvc/getallvms
   #+END_SRC

** 查看虚机的状态
   #+BEGIN_SRC sh
     vim-cmd vmsvc/power.getstate vmid
   #+END_SRC

** 打开虚拟机
   #+BEGIN_SRC sh
     vim-cmd vmsvc/power.on vmid
   #+END_SRC

** host 进入维护模式
   #+BEGIN_SRC sh
     vim-cmd hostsvc/maintenance_mode_enter
   #+END_SRC

** host 退出维护模式
   #+BEGIN_SRC sh
     vim-cmd hostsvc/maintenance_mode_exit
   #+END_SRC

* crontab 来配置定时任务
  直接配置关闭任务
  #+BEGIN_SRC sh
    vi /var/spool/cron/crontabs/root
  #+END_SRC
