* 参考命令实例
** 查看有哪些虚机
   #+BEGIN_SRC sh
     vim-cmd vmsvc/getallvms
   #+END_SRC

** 查看虚机的状态
   #+BEGIN_SRC sh
     vim-cmd vmsvc/power.getstate vmid
   #+END_SRC

** 打开虚拟机
   #+BEGIN_SRC sh
     vim-cmd vmsvc/power.on vmid
   #+END_SRC

** host 进入维护模式
   #+BEGIN_SRC sh
     vim-cmd hostsvc/maintenance_mode_enter
   #+END_SRC

** host 退出维护模式
   #+BEGIN_SRC sh
     vim-cmd hostsvc/maintenance_mode_exit
   #+END_SRC

* 配置 crontab  开机启动时自动添加
  配置开机启动时直接插入到 crontab 脚本中
  #+BEGIN_SRC sh
    vi /etc/rc.local.d/local.sh
  #+END_SRC

  添加的内容如下
  #+BEGIN_SRC sh
    /bin/echo '*/5  *    *   *   *   /esxi-scripts/probe.sh >/dev/null 2>&1' >> /var/spool/cron/crontabs/root
  #+END_SRC
